Your Smart Receipt Manager - Quick Start Guide

Getting Started

  1. Sign in using your Apple ID
  2. Start with a 60-day trial period to explore all features
  3. Begin uploading your receipts through the app

Adding Receipts

ExpenseIQ offers three convenient ways to add receipts:

  • Camera Scanning: Scan receipts directly using your camera
    • Ensure good lighting
    • Place receipt on a dark background
    • Hold your device steady while scanning
  • Upload Photos: Select existing receipt photos from your library
  • Manual Entry: Add receipt details manually if needed
Tip: After scanning, you'll receive a unique Receipt ID for tracking and can preview the scanned data before confirming.

Receipt Processing

The app automatically processes your receipts:

  • Automatic Data Extraction: The app reads store name, date, items, and amounts
  • Status Tracking: Monitor processing status (Pending, Complete) in the Overview
  • Verification: Green checkmarks indicate successfully processed receipts
Tip: If processing fails, you can retry scanning or manually edit the receipt details.

Managing Receipts

After adding receipts, you can:

  • Edit Details: Modify store information, date, items, and amounts
  • View Summaries: See subtotals, tax, and total amounts
  • Delete Receipts: Remove receipts if needed (action cannot be undone)
  • Sort and Filter: Organize by date, store, amount, or status

Understanding Your Spending

The Insights tab provides comprehensive spending analysis:

Time Period Analysis

  • View spending patterns for:
    • Past Week
    • Past Month
    • Past Year

Overall Statistics

  • Total Spent: Total amount across all transactions
  • Transactions: Number of receipts processed
  • Average Transaction: Average spending per receipt

Detailed Analytics

  • Top Stores by Spending: Visual breakdown of expenses by retailer
  • Top Items by Spending: Itemized list of purchases with quantities
  • Additional Insights:
    • Most Active Shopping Day
    • Average Items per Transaction
Tip: Use these insights to track spending patterns and identify major expenses over time.

Searching Receipts

Use the powerful search features to find specific receipts:

Search by Text

  • Enter any store name or item (e.g., "walmart", "groceries")
  • Search matches both store names and individual items

Date Search

  • Use date comparisons:
    • date > MM/DD/YY for dates after
    • date < MM/DD/YYYY for dates before
    • date = MM/DD/YY for exact date match
  • Supports both YY and YYYY date formats

Amount Search

  • Search by transaction amount:
    • total > 50 for amounts greater than
    • total < 100.50 for amounts less than
    • total = 49.99 for exact amount match
Tip: Combine search terms to narrow down results (e.g., search for "walmart" with "total > 50" to find larger Walmart purchases)

Navigation

Access key features through the bottom menu:

  • Overview: See all receipts in chronological order
  • Insights: Access spending analytics and trends
  • Upload: Add new receipts from photos
  • Camera: Scan new receipts directly
  • Settings: Manage app preferences and data

Managing Storage

The app efficiently manages your data with separate storage for:

  • Receipt images
  • Receipt data
  • Search cache
  • Insights cache
Tip: If you encounter any issues, try using the "Rebuild Cache" options in Settings.

Troubleshooting

Common solutions for issues:

  1. Receipt Processing Failed:
    • Ensure good lighting and contrast
    • Retake the photo using scanning tips
    • Try manual entry if scanning repeatedly fails
  2. Data Not Showing:
    • Use "Rebuild All Caches" in Settings
    • Check your internet connection
    • Verify the trial period hasn't expired
  3. Performance Issues:
    • Clear cache using appropriate rebuild options
    • Check available storage space
    • Update to the latest app version
Note: The "Reset All Data" option will delete all receipts and cannot be undone. Export your data before using this option.

Storage Management

ExpenseIQ manages several types of data:

  • Images: Original receipt photos
  • Receipt Data: Processed receipt information
  • Search Cache: For quick receipt lookup
  • Insights Cache: For analytics and reporting
Tip: Regular cache rebuilding helps maintain app performance. Use specific rebuild options in Settings for targeted maintenance.
